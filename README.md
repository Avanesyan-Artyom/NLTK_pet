# NLTK Sentiment Analysis

Проект по анализу тональности текста на базе **NLTK**. В основе — нескольких подходов (готовый лексикон, обучаемые модели, полноценный NLP‑пайплайн). Материал оформлен так, чтобы его можно было показать в резюме и быстро объяснить на собеседовании.

## Цели проекта

- Показать разные методы анализа тональности в NLTK: **VADER**, **Naive Bayes**, собственная подготовка признаков.
- Сделать воспроизводимый и читаемый код с аккуратной структурой.
- Подчеркнуть навыки работы с NLP‑пайплайном: токенизация → нормализация → признаки → обучение → оценка.

## Что реализовано

- **VADER** как быстрый baseline без обучения.
- **Субъективность (objective/subj)** на корпусе `subjectivity`.
- **Полярность (positive/negative)** на корпусе `movie_reviews`.
- Очистка текста, удаление стоп‑слов, лемматизация, POS‑теги.
- Вывод точности и информативных признаков для интерпретируемости.

## Быстрый старт

1. Установить зависимости:

```bash
pip install nltk
```

2. Открыть ноутбук:

```bash
jupyter notebook project3/NLTK_подробно.ipynb
```

3. В ноутбуке запустить первые ячейки — они скачивают датасеты NLTK:

- `punkt`
- `vader_lexicon`
- `stopwords`
- `subjectivity`
- `movie_reviews`
- `averaged_perceptron_tagger`
- `wordnet`

## Идеи для развития

- Добавить TF‑IDF или word embeddings.
- Попробовать Logistic Regression / SVM.
- Сделать визуализацию метрик (confusion matrix, ROC‑AUC).
